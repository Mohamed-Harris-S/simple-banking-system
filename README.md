# Simple Banking System

A console-based banking application built using **Java**, **JDBC**, and **SQLite**. The application simulates core banking operations such as account creation, authentication, balance management, and account deletion while storing data persistently in a SQLite database.

## Features

* Create a bank account
* Generate valid card numbers using the Luhn Algorithm
* Generate secure 4-digit PINs
* Login with card number and PIN
* Check account balance
* Add income to an account
* Delete an account
* Persistent data storage using SQLite
* Secure database access using JDBC Prepared Statements
