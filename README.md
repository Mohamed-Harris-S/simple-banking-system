# Simple Banking System

A console-based banking system built in Java.

## Features

- Create account
- Generate card number
- Generate PIN
- Login
- Check balance
- Luhn Algorithm for card validation

## Technologies

- Java
- IntelliJ IDEA

## Concepts Used

- OOP
- Encapsulation
- Classes & Objects
- Methods
- Random
- Loops
- Switch
- Luhn Algorithm

## Future Improvements

- SQLite Database
- Deposit & Withdraw
